## Installation

Before running the code, make sure you have the following dependencies installed:

- Python >= 3.6
- NumPy
- Pandas
- OpenCV (cv2)
- PyTorch
- Blender (if needed)

You can install these dependencies using the following command:

```bash
pip install numpy pandas opencv-python torch
